### Primary (Post to these people)
- **Freelancers** — designers, developers, writers, consultants
- **Founders** — solo founders and small team CEOs
- **Recruiters** — anyone booking 10+ interviews per week
- **Sales reps** — booking demos and discovery calls

### Secondary (These people share our content)
- Remote teams, productivity nerds, Google Workspace power users

### Psychographic
They are smart, time-poor, and sick of professional tools that create more friction than they solve. They tried Calendly, thought the free tier was too limited, and want something built for their workflow.

## 3. BRAND VOICE

| Dimension | Description | Example |
|-----------|-------------|---------|
| **Tone** | Confident but not arrogant | "We built this because the alternative was bad." |
| **Style** | Direct, concise, no fluff | Short sentences. No buzzwords. |
| **Personality** | Smart founder talking to peers | Not corporate. Not startup-bro. Real. |
| **Energy** | Calm urgency | We're moving fast but never desperate. |
| **Humour** | Dry and situational | Only when it earns the laugh. Never forced. |

### VOICE RULES
- ✅ Write like a person, not a press release
- ✅ Name the problem specifically before offering the solution
- ✅ Use "you" more than "we"
- ✅ Make claims specific ("saves 5 emails per booking") not vague ("saves time")
- ❌ Never say: "game-changing", "revolutionary", "disrupting", "synergy"
- ❌ Never use 3 exclamation points in one post
- ❌ Never open with "Exciting news!" — it's a skip trigger

## 4. VISUAL IDENTITY

### Color Palette (use these EXACT hex codes in all image prompts)
| Name | Hex | Use |
|------|-----|-----|
| **Brand Purple** | `#5B35E8` | Primary accent, CTAs, highlights |
| **Brand Blue** | `#3B6AE8` | Secondary accent, links |
| **Brand Cyan** | `#38BFFF` | Gradient end, highlights |
| **Deep Dark** | `#080c18` | Primary background (dark mode) |
| **Card Dark** | `#0f1628` | Secondary background, cards |
| **Text Primary** | `#e8edf8` | Headlines on dark bg |
| **Text Secondary** | `#7b8db8` | Supporting text on dark bg |
| **White** | `#ffffff` | Headlines on gradient bg |

### Brand Gradient
```
linear-gradient(135deg, #5B35E8 0%, #3B6AE8 55%, #38BFFF 100%)
```
Direction: top-left purple → bottom-right cyan

### Logo
The "D" lettermark in white, set on the brand gradient square with rounded corners (16px radius). Never stretch, recolor, or apply drop shadows to the logo.

### Typography
- **Headlines**: Bold, tight tracking, sentence case (not ALL CAPS)
- **Body**: Regular weight, generous line height
- **Accent labels**: All caps, wide tracking, small size

### Visual Aesthetic
Dark background tech aesthetic. Think: developer tools, Notion, Linear, Vercel. Not playful/colorful. Not corporate blue. Sophisticated, minimal, with subtle gradient glow effects.
